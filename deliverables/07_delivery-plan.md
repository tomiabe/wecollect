# Delivery Plan — What It Will Entail (End-to-End)

## Phase 0 — Alignment (1–3 days)

- Confirm primary audience + industries to prioritize
- Confirm primary conversion goal (recommended: Book a demo)
- Agree on sitemap and messaging hierarchy

## Phase 1 — UX + Content Strategy (3–7 days)

- Full heuristic audit (this folder)
- Page blueprints and copy framework
- Proof plan and content requirements checklist

## Phase 2 — Design + Build (4–6 weeks)

A single integrated sprint — design and production build run in parallel. Each page is designed and shipped iteratively so every completed section goes live as it's ready.

- High-fidelity pages for all primary pages (Home, Platform, Offerings, Solutions, Case Studies, Resources, Company, Contact)
- Component library and design system tokens
- Responsive variants (desktop, tablet, mobile)
- Site build (CMS optional)
- Analytics event tracking (demo requests, deck downloads)
- SEO basics and performance pass

## Phase 3 — Launch + Iteration (1–2 weeks)

- QA + content finalization
- Launch
- Monitor conversions + refine
- CTA optimisation based on early data

## Dependencies (what we need from WeCollect)

- Confirm offerings + packaging language (tool-only vs tool+people)
- Case studies + outcomes
- Security claims verification
- Sign off on proposed tech stack: custom build using latest web technology, Pages CMS (git-based) for content management, Cloudflare for hosting and edge performance
