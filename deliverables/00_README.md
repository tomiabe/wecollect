# WeCollect Website Revamp — Deliverables Index

End-to-end strategy for the WeCollect website revamp. Aligned to the CEO brief: **"Africa's native field process and data collection infrastructure"** — with a clear offering breakdown (Data Collection, M&E, Validation, Field Reporting) and two engagement models (Tool + People, or Tool only).

**The interactive strategy presentation lives at the root of this project** — `index.html`. Open it in any browser, or view it at the published GitHub Pages URL.

---

## Documents

| File | Contents |

| `01_executive-brief.md` | CEO brief distilled, primary audience, conversion goal, and success metrics |
| `02_current-site-audit.md` | Heuristic evaluation — 5 critical gaps across positioning, IA, credibility, CTAs, and design |
| `03_positioning-and-messaging.md` | Recommended category statement, messaging hierarchy, and tone principles |
| `04_information-architecture.md` | Proposed sitemap, navigation model, and before/after comparison |

| `07_delivery-plan.md` | Phase 0–3 timeline (Alignment → UX Strategy → Design + Build → Launch) and client dependencies |
